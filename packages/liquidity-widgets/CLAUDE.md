# Liquidity Widgets Package

Zap In widget - users add liquidity to concentrated liquidity pools using a single token or multi tokens. Handles token swapping, optimal routing, and liquidity provision in one transaction.

## Main Export

`LiquidityWidget` from `@kyberswap/liquidity-widgets` - see `src/index.ts` for full props interface.

## Demo

Run `cd apps/zap-widgets-demo && pnpm dev` for integration examples.
