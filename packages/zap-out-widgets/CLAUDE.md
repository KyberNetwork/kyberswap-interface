# Zap Out Widgets Package

Zap Out widget - users remove liquidity from concentrated liquidity pools and receive a single token. Handles LP token burning, token swapping, and optimal routing in one transaction.

## Main Export

`ZapOutWidget` from `@kyberswap/zap-out-widgets`
